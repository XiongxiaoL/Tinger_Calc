1. Instructions for use of probability theory

Input:

Enter the value corresponding to the probability in the value box (the first two rows).

Here you can choose to enter a single sample (a set of data) or a double sample (two sets of data)

eg: 2.3 0.8 12/5 3

Enter the probability value in the probability value box. The total probability value is 1, and the data length is the same as the sample length.

eg: 0.4 2/5 0.1 0.1

The K value is used to calculate the k-th order moment, which can be input if necessary

After inputting, click the "Confirm" button to calculate.

Output:

When a single sample is input, the expectation, variance, kurtosis coefficient, etc. are displayed at the bottom. If the k value is entered, the k-th origin moment and the k-th central moment are also displayed at the bottom.

Only when a double sample is input, the covariance and correlation coefficient of the sample will be output.

Note:

1.The "Whether to use equal probability" button has a higher priority than the probability data input box.

2.The decimal fraction can be recognized when entering, but the calculation process and display are all in decimal mode.

2. Part of the calculation formula

k-th central moment
Covariance
Correlation coefficient
